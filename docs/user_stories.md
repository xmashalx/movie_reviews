User stories

Browse site

*   Allow a user to see all movies on the website 
    
*   Allow a user to search through movies by search term
    
*   Allow a user to filter movies by director
    
*   Allow a user to filter movies by studio
    
*   Allow a user to filter movies by genres
    
*   Allow a user to click on movie and 
    

*   see details about the movie as well as reviews
    
*   Allow users to click on the user in any review to take them to the users profile
    

*   Allow a user to click on a list of all other users their names and bios
    
*   Allow users to click on user bios and
    

*   see their profile
    
*   read reviews they made 
    
*   Click on the movie name in the reviews they made and direct them to the movies page
    

Make a profile

*   Allow a user to click on register form
    

*   enter a username password and bio to create a profile
    

*   Once registered return to homepage
    
*   Allow user to view their own profile
    

Login/logout

*   Allow a user to login with details if they are registered and return to homepage 
    
*   Display an error message if user details not correct
    
*   Allow a user to be logged out of website if they click logout
    

Post Reviews

*   If user logged in:
    
*   Allow a user to browse movies, click on a movie and see the form to write a review
    
*   Allow a user to write a review with a rating and some text
    
*   Once review submitted it can be displayed on the movies own page or the users profile
